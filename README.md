# FindMeInTheCrowd

> A face search platform for events - Find yourself and your friends in event photos with ease!

Graduation Project of Thadoe Hein , Jessica Win 

##  About

FindMeInTheCrowd is an innovative platform that helps event attendees find their photos from large event galleries using facial recognition technology. Whether it's a concert, conference, festival, or party, our platform makes it easy to locate all the photos where you appear.

##  Public Repositories

Our platform consists of multiple components. Currently, the following web portals are open source:

### [Public Interface](https://github.com/FindMeInTheCrowd/public-interface-nextjs)
The user-facing web application where event attendees can:
- Browse event galleries
- Upload their photos for facial matching
- Find all photos where they appear
- Download and share their event photos

**Tech Stack:**
- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Radix UI Components
- React Hook Form with Zod validation

### [Admin Portal](https://github.com/FindMeInTheCrowd/admin-portal)
The administrative interface for event organizers and administrators to:
- Manage events and photo galleries
- Upload and organize event photos
- Monitor system usage and analytics
- Manage user access and permissions

**Tech Stack:**
- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Docker support
- Token-based authentication system
